Skills & workflow workbench

Author skills and Smithers workflows in natural language (pool writes the source, any tenant model can hold the pen), run workflows with pool executing every node, and watch eval results land live. Every run is a trajectory record.

Agents can drive all of this too: bun ui/bench.ts

The local API server is not running. Start it with bun ui/server.ts from the repo root, then reload.
