⌘K — quick action palette

type a slash command + what you want · enter · it builds in the background
/workflow map the fixture repo with $repo-map, scan deps in parallel, merge into a report| ↵ create
/skill read package.json and write a flat dep list to .laguna/dep-list.json with installed + latest versions| ↵ create
Enter starts generation in the background and takes you straight to the new workflow's canvas or skill's page — progress, graph verification, and the structure check land there. $skill-name autocompletes inline anywhere in the text.
