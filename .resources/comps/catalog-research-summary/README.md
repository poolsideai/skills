# Catalog Research Summary

This folder summarizes the comparative UI capture pass for skills catalogs, MCP registries, GPT stores, and agent/tool marketplaces.

- `capture-summary.json`: machine-readable capture log for each site and page.
- `design-findings.md`: synthesized UI patterns and recommendations for the Poolside Skills site.

Each captured site has its own sibling folder under `.resources/comps/` with screenshots, extracted rendered text, extracted headings/links/control metadata, and a per-site `README.md`.
