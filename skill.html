XS worker mode 
model_mode: laguna_xs_worker
task_type: single_file_patch
goal: Fix the failing parser edge case in one file.
constraints:
  max_files_to_modify: 1
  output_format: unified_diff
validator:
  type: command
  command: pnpm test parser-edge-case
repair_policy:
  max_repairs: 1
  return_only_corrected_output: true

Use when

  • A task needs to be handed to Laguna XS.2 as a narrow worker.
  • A router needs to choose between repo-map, CI log reduction, patching, or review skills.
  • The output needs schema, patch, command, or test validation before it is trusted.

Validation

SchemaContract validates against the task or router schema.
ScopeXS contracts cannot request broad planning or multi-file refactors.
RepairInvalid output gets one bounded correction attempt.
EscalationFailures report the blocking evidence instead of continuing loosely.
